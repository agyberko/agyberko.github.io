Chinese Foreign Direct Investment in Sub-Saharan Africa: Predicting Tiers of Investment¶

Abena Berko¶

Project Overview¶

Chinese has become one of the largest investors in Sub-Saharan Africa, particularly through the Belt and Road Initiative.

This project examines whether certain World Development Indicators (WDI) can predict the level of Chinese investment a country receives. Given a country's WDI profile, can we predict whether it receives negligible, low, moderate, or high Chinese investment?

Data Sources and ETL¶

My first dataset contains Chinese foreign direct investment (FDI) levels in all African countries, from 2004 to 2024. These totals include both state-owned enterprise and private firm investments. Included are levels of overall stock and annual flows, along with sector-specific totals. For now, I'll be focusing on the total annual FDI flows.

The focus of this project is Sub-Saharan Africa, so I'll be limiting my data to those countries (48 out of 54 countries).

Source: https://www.sais-cari.org/chinese-investment-in-africa

Some notes on cleaning: The WDI classifies Djibouti as Middle East/North Africa, so I'll be removing it from the FDI dataset.

Investment Tiers¶

Since the distribution is heavily skewed, defining "tiers" of investment may be more useful in identifying broad patterns of investment.

The tier thresholds below are roughly based on the distribution of raw FDI amounts across the entire sample (average being around $50M).

# Create tier labels
def assign_tier(fdi):
    if fdi <= 1:
        return 'Negligible'
    elif fdi <= 50:
        return 'Low'
    elif fdi <= 200:
        return 'Moderate'
    else:
        return 'High'

FDI_melted['FDI_tier'] = FDI_melted['FDI'].apply(assign_tier)

World Development Indicators¶

My next dataset is from the World Development Indicator (WDI) database.

Generally speaking, institutional quality, private sector regulations, and infrastructure quality tend to be linked to investment from both state-led and private firms (Lu and Nugent, 2021; William, Agyapong and Abass, 2013).

To reduce the amount of noise in the modeling process, the dataset will be restricted to the following categories:

  1. Economic/policy debt (GDP, national accounts, debt, etc.)
  2. Environment (natural resource rents)
  3. Infrastructure (railways, ICT, air transport)
  4. Private sector (business environment/regulations, international trade)
  5. Financial sector (banking access, mobile money, etc.)
  6. Public sector (resource distribution, governance)

Sources:

https://databank.worldbank.org/source/world-development-indicators#

Cantah, William Godfred and Wiafe, Emmanuel Agyapong and Adams, Abass (2013): Foreign direct investment and trade policy openness in Sub-Saharan Africa.

Jeffrey B. Nugent, Jiaxuan Lu, China's outward foreign direct investment in the Belt and Road Initiative: What are the motives for Chinese firms to invest?, China Economic Review, Volume 68, 2021.

# Standardize country names  (map WDI value to FDI value)
country_map = {
    'Cabo Verde': 'Cape Verde',
    'Gambia, The': 'The Gambia',
    'Somalia, Fed. Rep.': 'Somalia',
    'Central African Republic': 'CAR',
    'Sao Tome and Principe': 'Sao Tome & Principe',
}

# Replace ".." with NaN, standardize column names, remove metadata, pivot for merging
def clean_wdi(filepath):
    df = pd.read_excel(filepath, sheet_name=0)
    df = df.replace('..', np.nan)
    df = df.rename(columns={'Country Name': 'Country'})
    df['Country'] = df['Country'].replace(country_map)
    df = df[df['Country'].notna()].copy()
    df = df[~df['Country'].astype(str).str.contains('Data from database|Last Updated', na=False)].copy()
    year_cols = [c for c in df.columns if 'YR' in str(c)]
    long = pd.melt(df, id_vars=['Country', 'Series Name', 'Series Code'],
                   value_vars=year_cols, var_name='Year_raw', value_name='Value')
    long['Year'] = long['Year_raw'].astype(str).str.extract(r'(\d{4})').astype(int)
    long['Value'] = pd.to_numeric(long['Value'], errors='coerce')
    long = long.drop(columns=['Year_raw'])
    return long[long['Year'] <= 2024].copy()

# Load both WDI files
WDI_new = clean_wdi('WDI All variables.xlsx')              # 2012-2024
WDI_old = clean_wdi('WDI All variables 2004-2011.xlsx')    # 2004-2011

WDI_long = pd.concat([WDI_old, WDI_new], ignore_index=True)
WDI_long = WDI_long.drop_duplicates(
